# Nes>Tec - E-commerce de Componentes para PC

# Descripción del Proyecto

Nes>Tec es una aplicación web de e-commerce especializada exclusivamente en componentes para computadoras. Desarrollada como parte del programa Talento Tech, implementa las mejores prácticas de desarrollo web vistas en clase hasta a la fecha.

# Características Principales

#  Funcionalidades de E-commerce
- **Catálogo de componentes** especializado en PC
- **Carrito de compras** persistente con localStorage
- **Búsqueda y filtrado** por categorías específicas de componentes
- **Interfaz responsive** optimizada para todos los dispositivos

#  Diseño y Experiencia de Usuario
- **Diseño moderno** con Bootstrap 5 y CSS personalizado
- **Navegación fluida** con scroll suave
- **Animaciones y transiciones** para mejor UX
- **Modo carrito** deslizante para fácil acceso

# Tecnologías Implementadas
- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Framework CSS**: Bootstrap 5.3
- **Iconos**: Font Awesome 6.4
- **Fuentes**: Google Fonts (Roboto, Montserrat)


# Categorías Especializadas y Componentes Disponibles
- **Procesadores**: CPU AMD e Intel de última generación
- **Tarjetas Gráficas**: GPU NVIDIA y AMD para gaming y trabajo
- **Memorias RAM**: Módulos DDR4 y DDR5 de alta velocidad
- **Almacenamiento**: SSD NVMe y SATA de alta capacidad
- **Placas Madre**: Motherboards para diferentes sockets
- **Fuentes de Poder**: PSU certificadas 80 Plus
- **Gabinetes**: Cases con buen flujo de aire y diseño
- **Enfriamiento**: Coolers y disipadores eficientes
